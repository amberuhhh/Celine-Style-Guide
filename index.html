Celine Style Guide

This style guide for the Celine fashion brand website redesign encapsulates the brand's ethos of timeless elegance, sophistication, and modern minimalism. Through careful attention to typography, color, imagery, and user interface design, the revamped website aims to deliver a premium digital experience that resonates with Celine's discerning clientele, while setting new standards for luxury fashion in the digital realm.

Logo

Celines logo in the navigation and footer are to have a height of 2.4em.

Rules are to have a 1em clear padding around the logo.

Must never be another colour other than black and white.

Colour Pallette

    Pearl

  1. #F5F4EF
  2. rgb(245,244,239)
  3. cmyk(0,0,2,4)

The Pearl white would be used for slight background changes or focused items, such as the navigation and footer.

    Ink

  1. #1D1D1D
  2. rgb(29,29,29)
  3. cmyk(0,0,0,89)

The Ink black would be used for prominent actions, such as adding to shopping bag or purchase.

    Salmon

  1. #EC9FA7
  2. rgb(236,159,167)
  3. cmyk(0,33,29,7)

The Salmon pink would be used for secondary actions, such as signing up for the newspaper.

Typography

Heading 1's would be used sparringly, as a page title or product name. Use the Playfair Display SC Bold Font.

Heading 2's would be used as subheadings, introducing new sections. Use the Playfair Display SC Bold Font.

Heading 3's would be used for introducing subsections within a section.Use the Playfair Display SC Bold Font.

Heading 4's would be used as product titles on the home when in product cards (Shown on ). Use the Playfair Display SC Bold Font.

Heading 5's would be used for description headers, used within long informatin text. Use the Playfair Display SC Bold Font.

Heading 6's would be used for price, using the Playfair Display SC Bold Font.

Paragraphs would be used for long information texts, ingredients/descriptions. A line height of 1.4em, margin top & bottom 0.5em, using the Nanum Myeongjo Regular font.
